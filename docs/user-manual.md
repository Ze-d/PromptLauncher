# Prompt Launcher 使用手册

## 1. 安装

### 系统要求

- Windows 10 / Windows 11
- WebView2（Windows 10 通常已内置，如缺失会自动安装）

### 安装步骤

1. 下载安装包（`.msi` 或 `.exe`）。
2. 运行安装程序，按提示完成安装。
3. 安装完成后，Prompt Launcher 会自动启动并驻留在系统托盘中。

---

## 2. 快速开始

### 核心操作流程

```text
按 Ctrl + Alt + Space → 输入关键词搜索 → 上下键选择 → 回车复制 → 粘贴使用
```

这是最高频的使用路径，整个过程只需键盘即可完成。

---

## 3. 主窗口

双击托盘图标或从托盘菜单选择"打开主窗口"即可打开。托盘图标现在显示应用图标。

主窗口采用三栏布局：

| 区域 | 功能 |
|------|------|
| 左侧分组栏 | 按分组筛选 prompt（全部、收藏、最近使用、自定义分组）。全部视图下收藏项自动置顶。 |
| 中间列表 | 显示当前分组下的 prompt，支持搜索过滤 |
| 右侧编辑区 | 新增或编辑 prompt 的标题、内容、标签、分组 |

### 新增 Prompt

1. 点击顶部工具栏的"新增"按钮。
2. 在右侧编辑区输入标题和内容。
3. 可选：添加标签、选择分组、填写描述。
4. 点击"保存"。

### 编辑 Prompt

1. 在列表中点击要编辑的 prompt。
2. 在右侧编辑区修改内容。
3. 点击"保存"。

### 删除 Prompt

1. 在列表中选中要删除的 prompt。
2. 点击"删除"按钮。
3. 确认删除。

---

## 4. 快速搜索窗口

按 `Ctrl + Alt + Space` 唤起快速搜索窗口。

### 交互说明

| 操作 | 效果 |
|------|------|
| 输入文字 | 实时搜索匹配的 prompt |
| `↓` / `↑` | 上下移动选择 |
| `Enter` | 复制当前选中的 prompt 到剪贴板 |
| `Esc` | 关闭快速搜索窗口 |

搜索范围包括：标题、内容、描述、标签、分组名。标题匹配权重最高。

---

## 5. 系统托盘

Prompt Launcher 关闭主窗口后不会退出，而是在系统托盘中后台运行。

### 托盘菜单

右键点击托盘图标：

- **打开主窗口** — 显示主窗口
- **快速搜索** — 直接唤起快速搜索窗口
- **设置** — 打开 Settings Modal
- **退出** — 完全退出应用

---

## 6. 设置

在主窗口 Prompt 列表右上角点击齿轮 ⚙ 按钮，弹出 Settings Modal。

### 可配置项

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 主题 | System（跟随系统）/ Light（亮色）/ Dark（深色），即时切换 | System |
| 快捷键 | 唤起快速搜索窗口的全局快捷键。点击 Record 进入录制模式，按下组合键后点 Save 保存。 | `Ctrl + Alt + Space` |

修改快捷键后立即生效，如与其它软件冲突会在后台输出警告。

### 数据导入导出

Settings Modal 底部包含导入导出按钮：

- **Export JSON** — 导出全部数据（含 groups、tags、prompts）为 JSON 文件
- **Import JSON** — 选择 JSON 文件导入，自动创建不存在的分组和标签，重复标题追加 `(imported)` 后缀。结果报告包含导入的 prompts、groups、tags 数量。

---

## 7. 数据导入导出

在主页面点击齿轮 ⚙ 打开 Settings Modal，即可执行数据备份和恢复。

### 导出

点击"导出数据"，选择保存位置，所有 prompt（含标签和分组）会导出为 JSON 文件。

### 导入

点击"导入数据"，选择之前导出的 JSON 文件。导入时会校验数据格式，重复标题会自动追加后缀避免冲突。

---

## 8. 常见问题

### Q: 快捷键没反应？

1. 检查 Prompt Launcher 是否在托盘运行。
2. 检查快捷键是否被其他软件占用。
3. 在主页面点击齿轮 ⚙ → 点击 Record 重新录制新快捷键。

### Q: 数据存在哪里？

数据保存在本地 SQLite 数据库文件中，位于应用数据目录。不上传任何数据到网络。

### Q: 如何备份数据？

使用设置页的"导出数据"功能，导出为 JSON 文件保存即可。

### Q: 关闭窗口后程序还在运行？

这是设计行为。关闭主窗口 = 隐藏到托盘，程序继续在后台运行以保证快捷键可用。要完全退出，请从托盘菜单选择"退出"。
