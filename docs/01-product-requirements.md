# Prompt Launcher 产品需求文档

## 1. 项目概述

Prompt Launcher 是一个轻量级桌面效率工具，用于管理、搜索和快速复制常用 prompt。

用户可以将常用 prompt 保存到本地，通过全局快捷键唤起快速搜索窗口，输入关键词检索 prompt，按回车后将 prompt 内容复制到系统剪贴板，然后粘贴到 ChatGPT、Claude、Cursor、浏览器、IDE 或其他文本输入场景中。

第一版优先保证 Windows 可用，但技术架构预留 macOS 和 Linux 兼容空间。

------

## 2. 项目目标

### 2.1 核心目标

1. 快速保存常用 prompt。
2. 快速检索常用 prompt。
3. 通过全局快捷键唤起搜索窗口。
4. 回车后复制 prompt 到剪贴板。
5. 本地持久化保存数据。
6. 软件可后台常驻系统托盘。
7. 架构预留跨平台兼容能力。

### 2.2 用户价值

用户在使用 AI 工具时，常常会重复输入类似 prompt，例如：

1. 论文总结 prompt。
2. 代码审查 prompt。
3. 英文润色 prompt。
4. 周报生成 prompt。
5. 面试准备 prompt。
6. 翻译 prompt。
7. 架构设计 prompt。
8. Codex 任务 prompt。

Prompt Launcher 的价值是缩短这条路径：

```text
想起 prompt → 找历史记录 → 复制 → 修改 → 使用
```

变成：

```text
按快捷键 → 搜索 → 回车复制 → 粘贴使用
```

------

## 3. 产品定位

Prompt Launcher 不是完整 AI 客户端，不直接调用大模型 API，不做聊天功能，也不做复杂知识库。

它的定位是：

```text
本地优先的 prompt 快捷启动器。
```

核心关键词：

```text
轻量
快速
本地
可搜索
可快捷键唤起
可复制
低打扰
```

## 8. 核心用户流程

### 8.1 新增 prompt 流程

```text
打开主窗口
→ 点击新增
→ 输入标题
→ 输入内容
→ 添加标签
→ 选择分组
→ 保存
```

验收标准：

```text
新增后 prompt 出现在列表中。
重启软件后 prompt 仍然存在。
```

------

### 8.2 快速使用 prompt 流程

```text
用户正在使用任意软件
→ 按 Ctrl + Alt + Space
→ 快速搜索窗口弹出
→ 输入关键词
→ 结果列表实时过滤
→ 上下键选择目标 prompt
→ 回车
→ prompt 内容复制到剪贴板
→ 窗口关闭
→ 用户手动 Ctrl + V 粘贴
```

验收标准：

```text
从按快捷键到复制完成，整个过程可以只用键盘完成。
```

------

### 8.3 托盘运行流程

```text
用户点击关闭主窗口
→ 主窗口隐藏
→ 应用仍在托盘运行
→ 用户右键托盘图标
→ 可以重新打开主窗口或退出应用
```

验收标准：

```text
关闭窗口不等于退出应用。
用户必须从托盘菜单明确退出应用。
```

## 11. 搜索规则

MVP 搜索采用简单权重排序。

匹配范围：

1. 标题。
2. 标签。
3. 内容。
4. 描述。
5. 分组名。

推荐排序权重：

```text
标题命中：+50
标签命中：+30
描述命中：+15
内容命中：+10
收藏：+10
最近使用：+0 到 +20
使用次数：+0 到 +20
```

第一版可以先使用 SQLite LIKE 查询，再在应用层排序。

后续可升级为 SQLite FTS5 或本地模糊搜索索引。
